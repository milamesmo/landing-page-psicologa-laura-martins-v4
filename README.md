# Landing Page Profissional | Psicologa Laura Martins

Projeto ficticio de landing page profissional para uma psicologa clinica, criado para portfolio e estudo de interface, copy e direcao visual com apoio de IA.

## Sobre o projeto

A pagina apresenta uma profissional ficticia, seus formatos de atendimento, perguntas frequentes e chamada para contato via WhatsApp. A proposta visual busca uma estetica simples, organica e acolhedora, com paleta suave, imagens geradas por IA e estrutura responsiva.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Imagens geradas com IA

## Recursos

- Layout responsivo
- Menu mobile
- Hero com chamada principal
- Secao Sobre com imagem profissional ficticia
- Cards de atendimento
- FAQ com elemento `details`
- Botao flutuante de WhatsApp
- Estrutura semantica e SEO basico

## Observacao

Este projeto e totalmente ficticio. Nome, CRP, telefone, textos profissionais e imagens foram criados apenas para fins de estudo e portfolio.

## Como visualizar

Abra o arquivo `index.html` no navegador.
